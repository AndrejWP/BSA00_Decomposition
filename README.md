# BSA00_Decomposition

Учебный проект Школы 21 по базовым задачам IT-аналитика: анализу источников информации, составлению глоссария, декомпозиции предметной области, классификации объектов и описанию бизнес-правил.

## Описание проекта

В рамках проекта были выполнены аналитические задачи, связанные со структурированием предметной области и подготовкой аналитических артефактов для дальнейшего проектирования системы.

## Что было сделано

* Проанализированы исходные материалы предметной области.
* Выделены ключевые понятия, сущности, роли и ограничения.
* Составлен глоссарий терминов.
* Выполнена декомпозиция предметной области.
* Описаны основные типы объектов и их классификация.
* Сформулированы бизнес-правила.
* Подготовлена документация, которую можно использовать как основу для проектирования требований, API и базы данных.

## Полученные навыки

* Requirements Analysis
* Analytical Thinking
* Decomposition
* Change Management
* Business Rules
* Documentation
* Structured Analysis

## Результат

Проект помог закрепить навыки анализа требований, структурирования неформальной информации, выделения бизнес-правил и подготовки аналитической документации.
